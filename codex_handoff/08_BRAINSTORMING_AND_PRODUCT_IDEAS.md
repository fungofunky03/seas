# Brainstorming and Product Ideas

## Core product wedge

The best first wedge is not a full CRM. It is the workflow bridge between quote and crew.

Why:

- Owners feel pain when a lead becomes a job.
- Crews feel pain when sales notes are incomplete.
- Warranty/service depends on install documentation.
- This workflow is specific to permanent lighting and harder for generic CRMs to copy.

## Automation ideas

### Lead intake automation

When a lead comes in:

- Capture source.
- Ask for address/photos.
- Assign sales owner.
- Start quote checklist.
- Flag missing info.

### Quote preparation automation

When measurements are entered:

- Calculate material assumptions.
- Suggest controller/transformer count.
- Build proposal draft.
- Flag margin below target.

### Crew handoff automation

When quote is approved:

- Create crew job sheet.
- Create material pre-stage list.
- Generate homeowner confirmation.
- Add install checklist.
- Create service record stub.

### Service automation

When service issue is logged:

- Pull original install photos.
- Pull controller/transformer notes.
- Identify affected zone/channel.
- Create service ticket.
- Suggest annual tune-up upsell.
